# Submission Checklist

Перед тим як надсилати GitHub URL у форму Lapathoniia Ecosystem, перевірте:

- [ ] `lapathoniia.project.json` заповнений реальними даними.
- [ ] `name` короткий і придатний для публічної картки.
- [ ] `description_uk` пояснює, що робить проєкт.
- [ ] `website`, `demo` або `github` ведуть на публічну сторінку.
- [ ] `logo` веде на файл у цьому репозиторії.
- [ ] У репозиторії немає API keys, паролів або `.env` з production secrets.
- [ ] Email у `contact_email` є контактною адресою команди.
- [ ] Команда готова підтвердити заявку через beta email Lapathoniia.

## Logo

Рекомендації:

- SVG або PNG;
- прозорий фон;
- мінімум 512x512 для PNG;
- без дрібного тексту, який не читається в маленькому розмірі;
- файл до 1 MB.

## Cover

Обкладинка або скріншот не є обов'язковими, але допомагають краще показати
проєкт у вітрині екосистеми.

Рекомендований формат:

- 1200x630;
- PNG, JPG або SVG;
- без приватних даних на скріншоті.
